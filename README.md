# AE Utility Panel

A compact **Adobe After Effects ScriptUI panel** that provides quick-access tools for common workflow tasks like creating layers, camera rigs, keyframe alignment, precompose automation, and precomp decomposition.

Designed to speed up everyday compositing tasks inside After Effects.

---

## Features

### Layer Creation
- **Null** – Creates a null layer aligned with selected layer timing.
- **Adjustment Layer (ADJ)** – Creates an adjustment layer matching selected layer timing.
- **Solid** – Creates a solid using a color picker.
- **Text** – Quickly creates text layers.

### Utility Tools
- **1f** – Creates a **1-frame adjustment layer** at the current playhead.
- **Cam** – Creates a **camera with controller null rig**.

### Keyframe Tools
- **AK (Align Keys)** – Aligns selected keyframes sequentially by frame.

### Composition Tools
- **PreComp** – Precompose selected layers **individually** while preserving timing.
- **De** – Decompose selected **precomp layers** back into the main composition.

---

## Installation

1. Download the script.
2. Copy the `.jsx` file into:


After Effects/Scripts/ScriptUI Panels/


Example:


C:\Program Files\Adobe\Adobe After Effects\Support Files\Scripts\ScriptUI Panels


3. Restart After Effects.

4. Open the panel:


Window → AE Utility Panel


---

## Usage

1. Open any composition.
2. Select layers if required.
3. Click a button in the panel to execute the tool.

Most tools automatically match the **timing and position of selected layers**.
